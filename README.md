# Breast Cancer Prediction App

This application predicts whether a breast cancer tumor is malignant or benign using logistic regression.

## Installation

1. **Clone the repository:**

    ```bash
    git clone <repository_url>
    ```

2. **Install the required dependencies:**

    ```bash
    pip install -r requirements.txt
    ```

## Usage

1. **Run the Streamlit app:**

    ```bash
    streamlit run app.py
    ```

2. **Access the app in your web browser at `http://localhost:8501`.**

3. **Adjust the input parameters using the sliders in the sidebar.**

4. **Click the "Predict" button to get the prediction for the input data.**

## Components

- **app.py**: Contains the Streamlit application code.
- **requirements.txt**: Lists the Python dependencies required to run the application.
